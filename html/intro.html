Introduction

GamBit UI is an open source styling framework, which helps Developers to use best styles and components, by using pre-defined classnames. GamBit UI helps you to quickly build your Dream Project, by focusing much on Functionality, because we take care of your Designs.

Installation

In order to introduce GamBit UI to your project, Copy the Link given below and paste it in the < head > tag of your html, above all other style sheets.

Or you can paste this link at the top of your css file

Download

Download GamBit UI to get the compiled CSS and JavaScript, source code, or include it with your favorite package managers like npm, RubyGems, and more. and use GamBit UI in your projects.

Colors

GamBit UI comes with 40 different pre-defined colors , which are consistent in the overall design. The colors used are chosen on a generic basis, which can be accessed by pre-defined classes.

Different Shades

GamBit UI provides 4 differnet shades of each colors. You can change each and every color by just adding suffix -d for Dark, -l for Light, -xl for Extra Light to the specific color variables.

For code: use red in 4 different variatioins var(--red-xl), var(--red-l), var(--red), var(--red-d)
