Badge

Badges are small circles, positioned either at top-right or bottom-right of the parent component. Badge can be used to display numbers, online / offline status.

Profile Badge

Profile Badge is used to show the current status of a user . GamBit UI provides total 3 variants: online, idle and offline

Text Badge

Text Badge is used in notificatons using badge-text class
