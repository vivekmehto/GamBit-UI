Utilities

GamBit Ui mainly provides 2 types of Utilities: Text Utilities and Spacing Utilities.

Text Utilities

.text-lt for text line through, .text-ul for text underline, .text-up for text uppercase, .text-p-color for text primary color, .text-s-color for text secondary color, .text-grey for text grey color, .text-small for small text, .text-large for large text, .text-center for center text, .text-left for left text, .text-right for right text.

Spacing Utilites

GamBit UI uses Margin and Padding properties like

.m-05 for margin 0.5rem,

.m-01 for margin 1rem,

.m-15 for margin 1.5rem,

.m-2 for margin 2rem

.p-05 for padding 0.5rem,

.p-01 for padding 1rem,

.p-15 for padding 1.5rem,

.p-2 for padding 2rem
